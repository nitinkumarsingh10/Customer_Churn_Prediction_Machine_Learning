# Customer_Churn_Prediction_Machine_Learning
# 📊 Customer Churn Prediction

## 🚀 Project Overview

This project focuses on predicting whether a customer will leave a company (churn) using machine learning techniques. The goal is to help businesses improve customer retention by identifying at-risk customers.

## 📂 Dataset

* File: `Churn_Modelling.csv`
* Contains customer details such as:

  * Credit Score
  * Geography
  * Gender
  * Age
  * Balance
  * Estimated Salary
  * Tenure
  * Exited (Target Variable)

## ⚙️ Technologies Used

* Python 🐍
* Pandas, NumPy
* Matplotlib, Seaborn
* Scikit-learn

## 🔍 Workflow

1. Data Cleaning & Preprocessing
2. Encoding categorical variables
3. Feature scaling
4. Model training using ML algorithms
5. Model evaluation using accuracy and confusion matrix

## 🤖 Models Used

* Logistic Regression
* Random Forest
* Decision Tree

## 📈 Results

* Achieved good prediction accuracy on test data
* Identified key features affecting customer churn

## 💡 Key Insights

* Customers with low tenure are more likely to churn
* High balance customers show different churn behavior
* Geography plays a significant role
